# Terminal Tetris
Simple text UI tetris game for the terminal - UI based on the crossterm TUI library.

```
% cargo run
```

Controls: 
* q to quit
* Arrow Left & Arrow Right to move sideways
* Arrow Up to rotate
* Arrow Down to drop
* Space to pause
